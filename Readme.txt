酒店是一個神奇的場所

不地區也會有不同的稱號:『有些地方叫做夜總會、有些地方稱為俱樂部、因為一些稱為高級會館』

前幾個禮拜才會有某家知名酒店的小姐確診(武漢肺炎)的導致全台所有的酒店和相關產業都被迫歇業引起軒然大波

網路上有很多人再討論這件事然後

我就發現一件事就是好多女人搞不懂酒店對男人的吸引力

也幫助更多像我一樣清純的善男信女們

這次就要來跟各位聊一聊酒店究竟有什麼樣知道酒店消費的都是什麼樣的人來說可以分成5種:

第一種:是應酬類也就是為了談生意才去的人，就像我這種內心純潔被老闆硬被逼去的，我也是千百個不願意啊
第二種:是空虛累:酒店消費不便宜填滿內心的空洞還是該被小姐騙了傻子的
第三種:的是沒有別的目的老子就是想花錢當大爺(小姐最討厭這類的客人，除非他們真的很敢花錢)
第四種:同行也就是俗稱的男公關其實男公關，跑去酒店消費小姐跑去男公關店消費這些都是很正常的，因為一方面可以互相作業績，另外一方面的就是單純的發洩，因為平常男公關在上班的時候，都被那個深宮怨婦給蹂躪，所以偶爾也會想換個身份那也是很正常
第五種:就是研究類也就是純粹去各大酒店研究各種小姐的身體構造，非常的有實驗精神，這邊你可能有一問題:如果只是要研究的話，其實管道還有很多啊，為什麼一定要去酒店中高級場，所因為這一類型的人通常會覺得外面那種一兩千或兩三千不衛生，在他們的認知裡貴一點的，會比較乾淨就是這個邏輯

接下來要講的就是酒店到底哪裡吸引人:

第一點:超豪華 相信都應該都有跟朋友去唱ktv的經驗吧像台灣最有名的那個錢櫃不管是大廳，還是包廂裡面都是很豪華吧，接著請發揮想像力的程度乘上100倍，大概就是酒店的樣子，讓你宛如置身在天堂的錯覺，
第二點:需要隱密性比較高的人，在酒店進出可以過濾多閒雜人等，ktv會有那個學生拿在走廊上亂跑+鬼叫阿
第三點:就是你自己也會變得很豪華，例如你去ktv消費都叫你什麼先生嗎對不對，王就叫你王先生，在酒店 王先生變成王董 就是一個爽
第四點:有高度自主權，如果你有曾經在路邊花錢叫雞(肉飯)來吃的，就是當門口有好幾個小姐而且你有選擇障礙，不知道要選哪一個的時候，機頭會告訴你不用跳了啦，你在選老婆，天啊我今天的花錢都沒得選擇，還要被你酸，心裡都不是滋味啊，可是酒店的做法完全不一樣，他隨便都是10個小姐起跳，給你挑了，你都不喜歡有沒有關係再來10個，還是不喜歡，在來10個給你慢慢挑，挑到你滿意，完全沒有時間限制，你們知道為什麼嗎，因為選妃所花的時間，通常都會算在小姐的節數裡面
第五點:為什麼相談生意和應酬之類的事情都要在酒店進行，公司的會議是哪個不行，其實我也在酒店裡談生意識有玄機的，因為我們的人在喝完酒之後講話很實，就是你的本性會露出來，酒品會出賣的人品，你這個朋友值不值得深交，通常上一次酒店就知道，所以說不能在公司喝就是這樣，因為想看幾個高級主委會議室裡喝的酒+划拳脫衣服能看嗎，而要喝酒快炒店也可以喝，可是你要知道在商場上其實有很多秘密不能外流的，所以如果你們去快炒店談生意，搞不好隔壁坐著一個人他的媽媽的表舅的三叔公的小阿姨的同學剛好就是你們的競爭對手，這樣機密就會外洩，酒店包廂就成了最佳的場所
第六點:就是展示財力，若有人告訴你他家財萬貫，可是他卻一頓飯，都請不了你請問你會相信嗎，反之如果我跟你說他們公司光資本而就破億，但是卻連酒店這種一晚花的幾萬，都捨不得的時候，你心理對我們打算有大問號，所以去酒店的目的就是要讓對方看到我公司不在呼這小錢，如果你今天帶著客戶去酒店，一進去經理忙跟你說，哎呦自己為什麼幾乎天天看到你，最近生意很好喔，而且一進來包廂服務小費都好幾台再給的做法，一定會比你拿出公司年度財務報表，還有更有用
第七點:酒店裡談生意的成功機率，可是比在公司裡還有高出許多，最大的工程就是酒店小姐，聰明的酒店小姐會知道要察言觀色，懂得利用形式來推播出聲，因為在酒店小姐來看你們談什麼與我無關，但我知道一個重點就是如果談成的客人心情好，那麼今晚就一定有賺頭
第八點:酒店小姐的做法和老婆不一樣，它讓產生一種幻覺，你會感覺你是個很完美且零缺點的男人，例如你個子不高只有160幾公分，在外面很多女生都看不上你，但是當你跟酒店小姐訴苦時她會告訴你，重點是要自信，有自信的男人才是最帥的，而且會告訴你矮的男人通常比較會賺錢，總之不管你身上有什麼樣的缺點，他都可以當作沒看見，就算的熱狗就有3公分，19秒鐘，硬度就跟蒟蒻沒有兩樣，他都可以演出你這輩子從沒看過最爽的那個表情，然後當你發現酒店小姐有什麼樣的缺點，希望他能去改的時候，他會虛心接受，然後跟你說為了你我一定會改掉的，那你以後一定要常常來陪我喔，所以你會發現男人很容易掉進那樣的陷阱

我問我老婆:為什麼酒店小姐為了錢就演戲，而我的不管是白天的工作還是這個頻道的所以我都全數上繳，但我還是沒有享受到酒店待遇，她直接告訴我，酒店小姐一節多少錢我告訴他大概200到300左右吧，然後買小姐出場要多少錢，我跟他說像你這貨色大概8000左右吧，那個是買短鐘啦，不過夜的，如果是買長的話，通常是兩萬起跳，結果他聽完後，直接算給我看，他說我平日都是晚上6點到家，到晚上12點之前是6個小時，以就是36節，一節如果算我250塊錢就，好那麼總數就是9000塊，然後從晚上12點到早上6點，算是買他的長鐘，也就是20000塊，就來說平日一天就要29000塊，假日的話扣掉過夜6個小時，一天總要給我18個小時，也是108節，在加長鐘47000，如果把平日22天的29000塊，再加上假日8天47000塊的話，等於每個月要付她101萬又4000塊，打8折老娘半買半相送，且零頭不跟我算，一個月也要80萬
